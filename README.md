# 📝 Smart To-Do List App

A simple and interactive To-Do List web application built using HTML, CSS, and JavaScript.

##  Features
- Add tasks
- Delete tasks
- Mark tasks as completed
- Filter tasks (All / Completed / Pending)
- Saves tasks using localStorage

##  Technologies Used
- HTML
- CSS
- JavaScript

##  How to Use
1. Enter a task
2. Click Add button or press Enter
3. Manage your tasks easily!

## Future Improvements
- Edit task feature
- Dark mode
- Cloud storage
